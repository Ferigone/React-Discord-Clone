# ReactJS Discord Clone

React Discord clone made with ReactJS and Firebase

## Installation

Use the package manager [NPM/NodeJS](https://nodejs.org/en/download/) to install Discord clone.

```bash
npm install
```

## Usage

Configure firebase

```python
/utils/firebase.js
```
